## Структура модулей проекта 📂

Ниже представлено описание каждого файла, входящего в состав проекта, и его роль в обработке транзакций:

### 📁 src/ (Исходный код)
- **`main.py`** — Центральный модуль управления. Реализует логику взаимодействия с пользователем, последовательную фильтрацию данных и итоговый вывод результатов в консоль.
- **`utils.py`** — Набор вспомогательных инструментов для работы с JSON-файлами, фильтрации по валюте и реализации поиска по описанию с помощью регулярных выражений (`re`).
- **`processing.py`** — Содержит логику аналитической обработки: фильтрацию по статусу (`EXECUTED`/`CANCELED`), сортировку по дате и подсчет количества операций по категориям через `Counter`.
- **`masks.py`** — Реализует низкоуровневые алгоритмы маскировки: скрывает часть цифр в номерах карт и банковских счетов для обеспечения безопасности.
- **`widget.py`** — Высокоуровневый модуль-обертка. Определяет тип платежного средства (карта/счет) для применения маски и конвертирует даты в формат `ДД.ММ.ГГГГ`.
- **`widget_csv.py`** — Модуль интеграции с `pandas` для обработки CSV-файлов. Обеспечивает чтение таблиц и автоматическую маскировку данных прямо при загрузке.
- **`widget_excel.py`** — Аналог модуля для CSV, оптимизированный для работы с файлами Excel (`.xlsx`). Использует движок `openpyxl` для корректного чтения данных.
- **`generators.py`** — Содержит итераторы и генераторы для эффективной работы с большими объемами данных без перегрузки оперативной памяти.
- **`external_api.py`** — Отвечает за связь с внешним API курсов валют для автоматической конвертации сумм из USD/EUR в RUB.
- **`decorators.py`** — Содержит декоратор `@log` для мониторинга функций. Фиксирует время запуска, аргументы, результат и возникшие ошибки в лог-файл или консоль.

### 📁 tests/ (Тестирование)
- **`test_main.py`** — Набор тестов для проверки пользовательского интерфейса и интеграционной логики программы с использованием `unittest.mock`.
- **`test_utils.py`** — Модульные тесты для проверки функций поиска, работы с регулярными выражениями и загрузки JSON.
- **`test_masks.py`** — Тесты для проверки корректности форматов маскировки (форматы `XXXX XX**` и `**XXXX`).
- **`test_processing.py`** — Проверка корректности сортировки и фильтрации данных по заданным критериям.
