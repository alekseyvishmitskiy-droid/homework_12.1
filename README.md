# Banking Operations Project

Проект для обработки и анализа финансовых транзакций. Позволяет фильтровать, сортировать и маскировать данные, а также работать с внешними API для конвертации валют.

## Описание модулей

### utils
Реализует функцию для загрузки данных из JSON-файла. Принимает путь до файла и возвращает список словарей. Если файл пустой, содержит не список или не найден, возвращает пустой список.

### external_api
Содержит функцию для получения суммы транзакции в рублях (тип `float`). Если операция проведена в USD или EUR, обращается к **Exchange Rates Data API** для конвертации по текущему курсу. Для безопасности использует переменные окружения из `.env`.

### masks
Базовый инструмент для скрытия конфиденциальных данных:
- **get_mask_card_number**: Маскирует 16-значный номер карты в формат `XXXX XX** **** XXXX`.
- **get_mask_account**: Маскирует номер счета в формат `**XXXX` (только последние 4 цифры).

### widget
Высокоуровневый обработчик входящих строк:
- **mask_card_and_account**: Автоматически определяет тип данных (карта или счет) и применяет соответствующую маску.
- **get_date**: Преобразует дату из формата ISO (2024-03-11T12:00:00) в формат `11.03.2024`.

### processing
Логика для работы с массивами транзакций:
- **filter_by_state**: Фильтрует операции по статусу (`EXECUTED`, `CANCELED`).
- **sort_by_date**: Сортирует транзакции от новых к старым (по убыванию даты).

### generators
Набор эффективных итераторов для больших массивов данных:
- **filter_by_currency**: Выдает транзакции только в заданной валюте.
- **transaction_descriptions**: Поочередно возвращает текстовое описание каждой операции.
- **card_number_generator**: Генерирует номера карт в формате `XXXX XXXX XXXX XXXX` в заданном диапазоне.

### decorators
Инструменты для мониторинга кода:
- **log**: Декоратор, фиксирующий имя функции, результат выполнения или тип ошибки с входными параметрами. Поддерживает вывод в консоль или запись в файл через аргумент `filename`.

## Настройка и запуск

1. Создайте файл `.env` на основе `.env.template` и укажите ваш API-ключ.
2. Разместите файл с данными `operations.json` в директории `data/`.

- **get_date**: Преобразует дату из формата ISO (2024-03-11T12:00:00) в формат `11.03.2024`.
